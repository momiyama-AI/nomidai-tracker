# Claude Code 開発規約

このファイルは Claude Code 向けの作業規約です。実装前に必ず `SPEC.md` と本ファイルを読んでください。

## 最重要ルール

- `SPEC.md` がこのプロジェクトの単一の真実です。迷ったら `SPEC.md` に従ってください。
- 仕様が曖昧な場合は、`SPEC.md` 末尾の推奨仮決めを採用し、必要な質問を報告してください。
- フェーズごとにスコープを守ってください。先のフェーズの機能を作り込まないでください。
- 各フェーズ完了時にビルド確認とテスト確認を行ってください。
- 各フェーズ完了時に必ず commit してください。
- コミットメッセージは日本語1行にしてください。
- SwiftData マイグレーションを壊す変更は、実装前に必ず人間へ事前申告してください。

## 実装方針

- Swift 5.10+ / SwiftUI / iOS 17.0+ を前提にしてください。
- SwiftData を使ってください。
- アーキテクチャは MV + Repository 層です。
- 過剰な抽象化、大きな DI コンテナ、外部ライブラリの追加は禁止です。
- サーバー、外部API、アカウント登録、ログイン機能は追加しないでください。
- 金額は必ず `Int` 円で扱ってください。金額計算に `Double` / `Decimal` を使わないでください。
- 度数は `abvTenthsPercent` の整数で扱ってください。
- 純アルコール量は `SPEC.md` の整数計算方針に従ってください。
- 日付と月境界は `Calendar.current` を使い、ユーザーのタイムゾーン基準にしてください。
- 文字列は `Localizable.strings` に集約してください。
- 画面を追加する場合は `#Preview` を用意してください。

## App Store 審査セーフティ

- アルコールを扱うため、年齢制限 17+ 前提です。
- 健康改善、医療効果、節酒効果、禁酒効果、依存症改善を示唆しないでください。
- 純アルコール量は参考情報としてのみ扱ってください。
- マスコットのセリフで説教、健康注意、飲酒量への評価をしないでください。
- StoreKit 実装時は購入復元、利用規約リンク、プライバシーポリシーリンクを必ず置いてください。
- Privacy Manifest はデータ収集ゼロとして整備してください。

## フェーズ運用

各フェーズで以下を実行してください。

1. `SPEC.md` と本ファイルを読む。
2. 指示されたフェーズ範囲だけを実装する。
3. 必要なテストを追加する。
4. `xcodebuild` などでビルド/テスト確認する。
5. 結果を短くまとめる。
6. 日本語1行で commit する。

## 禁止事項

- `SPEC.md` にない外部API、サーバー、ログイン機能を追加すること。
- 健康改善や節酒効果を示唆する文言を入れること。
- 金額計算に浮動小数点を使うこと。
- SwiftData モデルの保存項目を無断で削除/リネームすること。
- 別フェーズの有料機能や Widget を先行実装すること。
- ユーザーの未コミット変更を勝手に戻すこと。
