### サブタイトル

### プロモーション文

```text
家飲みも外飲みも、今月いくら使ったかをすぐ確認できる支出トラッカーです。予算、カレンダー、ウィジェット、のみだぬきの表情で飲み代のペースを楽しく把握できます。
```

### 説明文

```text
飲み代トラッカーは、家飲み・外飲みの支出をすばやく記録し、今月の飲み代をわかりやすく確認するためのアプリです。

主な機能:

- 缶ビール350ml/500ml、缶チューハイ、日本酒1合、ワイングラス、外飲みなどのプリセット記録
- 今月の支出合計、先月同日比、家飲み/外飲み内訳
- 月次予算と残額、消化ペース
- 日別支出のカレンダーヒートマップ
- マスコット「のみだぬき」による支出ペース表示
- 小/中サイズウィジェット
- Pro機能: 全期間グラフ、CSV出力、カスタムプリセット無制限、中サイズウィジェット

本アプリは完全ローカル動作です。サーバー、外部API、アカウント登録は使用しません。
```

### キーワード

```text
飲み代,家飲み,外飲み,支出,予算,家計簿,お酒,カレンダー,ウィジェット,CSV
```

## App Review Information

### Sign-in Required

```text
No
```

### Notes

```text
本アプリは、家飲み・外飲みの支出を端末内で記録し、今月の飲み代、予算、日別支出、ウィジェットを確認するためのローカル動作アプリです。

ログイン、アカウント登録、外部API、開発者サーバー連携はありません。審査用のデモアカウントは不要です。

記録データは端末内のSwiftDataに保存されます。Widget表示用の集計スナップショットのみ、App GroupのUserDefaultsに保存します。開発者サーバーへの送信や広告トラッキングは行いません。

アプリ内課金はStoreKit 2を使用しています。Pro機能は、全期間グラフ、CSV出力、カスタムプリセット無制限、中サイズウィジェットです。購入復元ボタン、利用規約、プライバシーポリシーへのリンクをペイウォールに設置しています。月額プランは1か月ごとの自動更新サブスクリプションであり、その旨と解約方法をペイウォール上に表示しています。

本アプリは酒類に関する支出記録アプリであり、飲酒を推奨するものではありません。健康改善、節酒、禁酒、依存症治療、医療助言を目的としません。日本国内では20歳未満の飲酒が法律で禁止されている旨を、利用規約および設定画面に表示しています。
```

## 審査時の確認手順

```text
1. アプリを起動する。
2. ホーム画面からクイック記録を開く。
3. 標準プリセットを選び、金額と数量を確認して記録する。
4. ホーム画面で今月合計、家飲み/外飲み内訳、予算、のみだぬき表示が更新されることを確認する。
5. カレンダー画面で日別支出と日別記録一覧を確認する。
6. 設定画面で月次予算、基準値、プリセット編集、利用規約、プライバシーポリシー、サポートリンクを確認する。
7. Pro導線からペイウォールを開き、商品表示、購入復元、法務リンクを確認する。
8. Widgetを追加できる環境では、小ウィジェットに今月合計とキャラが表示されることを確認する。中ウィジェットはPro機能です。
```
