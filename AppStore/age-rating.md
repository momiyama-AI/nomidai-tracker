# Age Rating Answers

最終更新日: 2026年7月3日

App Store Connect の年齢制限入力用メモです。

## 推奨

- 年齢制限: 17+
- 理由: アルコールへの言及があるため。

## 入力時の考え方

| 項目 | 回答案 |
|---|---|
| アルコール、タバコ、薬物への言及または使用 | あり |
| 頻度 | アプリ全体の題材として酒類支出を扱うため、必要に応じて高めに設定 |
| 医療/健康情報 | なし |
| ギャンブル | なし |
| 暴力表現 | なし |
| 性的表現 | なし |
| ユーザー生成コンテンツ | なし |
| Webアクセス | 法務/サポートリンクのみ |

## 審査向け補足

本アプリは酒類の支出記録アプリであり、飲酒を推奨するものではありません。健康改善、節酒、禁酒、依存症治療、医療助言を目的としません。日本国内では20歳未満の飲酒が法律で禁止されている旨を利用規約と設定画面に表示しています。
