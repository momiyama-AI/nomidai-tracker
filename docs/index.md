# 飲み代トラッカー

飲み代トラッカーは、家飲み・外飲みの支出を端末内で記録し、今月の飲み代を確認するためのiPhoneアプリです。

## 重要なリンク

- [利用規約](/nomidai-tracker/terms/)
- [プライバシーポリシー](/nomidai-tracker/privacy/)
- [サポート](/nomidai-tracker/support/)

## データの扱い

本アプリは、サーバー、外部API、アカウント登録機能を使用しません。記録データは原則として利用者の端末内に保存されます。
