# 飲み代トラッカー 利用規約

最終更新日: 2026年7月3日

本規約は、飲み代トラッカー（以下「本アプリ」）の利用条件を定めるものです。本アプリを利用することで、本規約に同意したものとみなします。

## 1. 本アプリの目的

本アプリは、酒類に関する支出、飲酒日の記録、予算、プリセット、参考値としての純アルコール量を、利用者自身の端末内で管理するためのアプリです。

本アプリは医療、健康改善、節酒、禁酒、依存症治療その他の専門的助言を提供するものではありません。

## 2. 年齢および法令遵守

日本国内では、20歳未満の飲酒は法律で禁止されています。本アプリは20歳以上の利用者による利用を想定しています。

飲酒運転その他の違法行為は行わないでください。利用者は、自身の責任において適用される法令を遵守するものとします。

## 3. データの保存

本アプリの記録データは、原則として利用者の端末内に保存されます。サーバー、外部API、アカウント登録による同期機能はありません。

端末の故障、紛失、アプリ削除、OS操作等によりデータが失われる場合があります。重要な情報は、必要に応じて利用者自身で控えてください。

## 4. 有料機能

本アプリでは、App Storeのアプリ内課金を通じて有料機能を提供する場合があります。有料機能の価格、内容、支払い、解約、返金等は、Appleの定める規約およびApp Storeの仕組みに従います。

購入済みの権利は、アプリ内の購入復元機能から復元できます。

## 5. 禁止事項

利用者は、本アプリの利用にあたり、以下の行為を行ってはなりません。

- 法令または公序良俗に反する行為
- 本アプリの改変、解析、過度な負荷をかける行為
- 第三者の権利を侵害する行為
- その他、本アプリの提供者が不適切と判断する行為

本アプリは、正確性、完全性、有用性、特定目的への適合性を保証するものではありません。

本アプリの利用または利用不能により利用者に生じた損害について、法令上認められる範囲で責任を負いません。
