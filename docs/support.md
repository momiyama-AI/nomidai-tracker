# 飲み代トラッカー サポート

最終更新日: 2026年7月3日

飲み代トラッカーに関する不具合報告、利用方法の確認、購入復元に関するお問い合わせは、App Storeの配信者情報またはサポート窓口からご連絡ください。

お問い合わせ時には、以下の情報を添えていただくと確認がスムーズです。

- iPhoneの機種名
- iOSのバージョン
- アプリのバージョン
- 発生した画面と操作内容

本アプリは完全ローカル動作のため、記録データは開発者側では確認できません。必要な場合は、利用者ご自身の判断で画面内容やCSV出力を共有してください。
